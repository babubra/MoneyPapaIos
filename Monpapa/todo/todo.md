# MonPapa — TODO

## 🧠 Умное интерактивное табло на дашборде

**Расположение:** под блоком «Последние операции» на главном экране (DashboardView).

**Суть:** Динамический блок, который показывает самую актуальную информацию
по приоритетной логике (не случайная ротация!). Пользователь также может свайпать
между доступными карточками (горизонтальный PageView с точками-индикаторами).

### Приоритеты отображения

| Приоритет | Условие | Карточка |
|:---------:|---------|----------|
| 🔴 1 | Есть непрочитанная подсказка | 💡 Подсказка / финансовый совет |
| 🟠 2 | Есть просроченные долги | 🤝 Список просроченных долгов |
| 🟡 3 | Прошла неделя → есть данные | 📊 Мини-график расходов за 7 дней |
| 🟢 4 | Есть транзакции за месяц | 🏆 Топ-3 категории расходов (с прогресс-барами) |
| ⚪ 5 | Нет данных / новый пользователь | 💡 Онбординг: «Добавьте первую транзакцию!» |

### Примеры подсказок (приоритет 1)

- «Сегодня вы не внесли ни одной транзакции. Действительно ничего не было?»
- «Вы тратите на кафе на 30% больше, чем в прошлом месяце»
- «Поздравляем! Расходы за эту неделю ниже средних 🎉»

### Технические заметки

- Реализовать как `SmartInsightView` — отдельный компонент
- Использовать `TabView` с `.tabViewStyle(.page)` для свайпа между карточками
- Данные считать из SwiftData-запросов прямо в View (через @Query)
- Подсказки хранить как enum с условиями срабатывания

### ⚠️ Необходимые изменения в моделях

- **DebtModel** — добавить поле `dueDate: Date?` (срок возврата долга)
  для поддержки функции «просроченные долги»

### UX принципы

- Контент предсказуем: пользователь понимает «тут самое важное прямо сейчас»
- Свайп даёт контроль: можно просмотреть все доступные карточки
- Точки-индикаторы (●○○) показывают количество карточек
