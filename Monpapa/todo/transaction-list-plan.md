## Контекст

**Цель:** Полноценная вкладка «Транзакции» с фильтрацией, группировкой по дням, поиском и итогами.

**Экран вызывается:** из TabBar (вкладка «Транзакции»)

**Зависимости (уже есть):**
- `TransactionModel` — данные
- `TransactionRow` — строка транзакции (переиспользуем)
- `TransactionDetailView` — тап → sheet с деталями
- `CategoryPickerView` — основа для фильтра по категориям
- `hideAmounts` — privacy mode
- Дизайн-токены: `MPColors`, `MPTypography`, `MPSpacing`, `MPCornerRadius`

## Примечания

- Все данные через `@Query` — реактивное обновление при изменениях
- `TransactionRow` уже поддерживает `hideAmounts`
- `TransactionDetailView` уже поддерживает inline-edit + delete
- Сортировка в списке: по `transactionDate` (дата операции), не `createdAt`
